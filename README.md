## About

This project is a basic version of Instagram App.
* Create user
* Upload user image
* Upload post
* Add comment
* Follow friends
* Like posts
* Message user
* Basic notifications
* Search user
* Share

## Tech I Use

* SwiftUI
* Firebase Auth
* Firestore
* Firebase Storage
* Kingfisher
* Async/Await
* MVVM
* Lottie Animations
